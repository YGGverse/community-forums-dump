# community-forums-dump

SQL tables dump and attachment files to set up independent community forum ([Flarum v2](https://flarum.org)) node.

Includes following (public-safe) tables only:

* `discussion_tag`
* `discussion_user`
* `discussions`
* `posts`
* `tags`

And the attachments are located in folder:

* `/public/data`
